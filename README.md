# 🚀 GitHub Contribution Visualizer

**3D animated GitHub profiles - Live data!**

## ✨ [Live Demo](https://startling-youtiao-1f3316.netlify.app)

## 📱 Features
- ✅ Real GitHub API (stars, repos, live data)
- ✅ 3D cube animations (hover effects)
- ✅ Mobile responsive (phone perfect)
- ✅ Works for any username
- ✅ Modern TailwindCSS design

- ## 🛠️ Tech Used
- HTML5 + CSS3 (3D) + JavaScript TailwindCSS + GitHub API Netlify Deploy

- ## 🎮 How to Use
1. Enter GitHub username (like `torvalds`)
2. Click "Visualize"
3. Watch 3D magic happen! ✨
